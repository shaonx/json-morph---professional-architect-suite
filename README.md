## ✨ 核心神通 | Core Features

### 🚀 GB 级流式解析 (GB-Scale Streaming)
- **拒绝 OOM**：采用先进的流式状态机解析技术，轻松应对 GB 级别的超大 JSON 文件。
- **Web Worker 并行**：解析过程在后台线程进行，界面丝滑不卡顿，解析进度实时感知。

### 🛡️ 绝对安全隐私 (Privacy First)
- **100% 本地化**：所有数据处理均在您的浏览器本地完成，绝不上传任何字节。
- **安全护身符**：内置隐私承诺面板，透明公开，保护您的商业机密与敏感信息。

### 🎨 极致交互体验 (Professional UX)
- **可视化树状视图**：支持大规模节点的延迟渲染（Lazy Rendering），300KB+ 数据秒开。
- **多维转换**：一键转换至 CSV, Excel, XML, YAML, TypeScript 类型定义等多种格式。
- **智能搜索**：高性能正则匹配与高亮，快速定位数据。
- **全能编辑器**：支持可视化节点编辑，轻松构建复杂 JSON 结构。

### 🌍 全球化支持 (Global Ready)
- 完美适配简体中文、繁体中文、英语、日语、韩语。

## 🚀 本地开发 | Local Development

想要在自己的洞府里运行？简单几步即可：

1. **克隆仓库**:
   ```bash
   git clone https://github.com/shaonx/json-morph---professional-architect-suite.git
   ```

2. **安装依赖**:
   ```bash
   npm install
   ```

3. **启动开发服务器**:
   ```bash
   npm run dev
   ```

4. **构建生产版本**:
   ```bash
   npm run build
   ```

## 🍮 布丁的小贴士 | Pudding's Tips

- **大文件建议**：处理超过 100MB 的文件时，请使用侧边栏的“流式解析”模式，布丁会帮你稳稳地守住内存！
- **快捷键**：善用搜索（`Ctrl+F`）和快速复制，能让你的架构效率翻倍哦！
